## ğŸš€ Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Basic knowledge of HTML, CSS, and JavaScript (for modifications)
- Git (for version control)

### Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/jawadabbasi14/Mj-Porfolio.git
   cd Mj-Porfolio
   ```

2. **Open locally**
   ```bash
   # Option 1: Open directly in browser
   open index.html
   
   # Option 2: Use a local server (recommended)
   python -m http.server 8000
   # Then visit: http://localhost:8000
   ```

3. **Customize the content**
   - Edit `index.html` for main content
   - Modify `css/style.css` for styling changes
   - Update `js/main.js` for functionality tweaks
   - Replace images in `assets/images/` with your own
